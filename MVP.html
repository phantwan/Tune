NFC Cards with RFID Module

When the user presses the "save" button, the music will be saved to an NFC card in an instant by the RFID module in the device, which can also read the card to playback the music without delay. The data stored in the card takes the format of the song ID, matching the music file from the database. Plus, the NFC card is compatible with almost all the mainstream mobile phones, making it possible for the younger generations to play the in-card music with their phones, a way to facilitate communication among the generations. The music in a card is not able to be rewritten due to the consideration of the elderly being more prone to mistake, also because the song relates to a special memory of them, which they do not wish to change or wipe away.

Physical prototype

In the physical prototype, we focus more on the user experience, so decide to replace the ESP-32 board which provides internet connection to an Arduino UNO + MP3 module to create the music experience.
